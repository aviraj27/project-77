PROJECT CONSOLE

Visual Studio Code is an IDE platform. Visual Studio Code, also commonly referred to as VS Code, is a source-code editor made by Microsoft with the Electron Framework, for Windows, Linux and macOS. Getting Started:
Open Visual Studio IDE software or download and install it.
Create a new folder on your computer.
Open Visual Studio IDE and click on File > Open Folder, then browse and open the folder created in step 2.

In console screen we can test our code or simply it is the screen where we can do our rough code.
To open console screen we right click on the web page and select inspect option, a pane will appear on the right side.
On the top right section of the pane we can see the the console button beside the elements button.
Click on it and console pane will open.
We use "console.log()" to print anything in the console pane.
